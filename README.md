## 数据结构

### 线性表

#### 数组

##### 特点

* 相同类型的数据元素
* 连续的内存空间

##### 操作时间复杂度

| 操作    | 时间复杂度 |
| ------- | ---------- |
| prepend | O(n)       |
| append  | O(1)       |
| lookup  | O(1)       |
| insert  | O(n)       |
| delete  | O(n)       |

#### 链表

##### 特点

* 相同类型的数据元素
* 可以是不连续的内存空间

##### 操作时间复杂度

| 操作    | 时间复杂度 |
| ------- | ---------- |
| prepend | O(1)       |
| append  | O(1)       |
| lookup  | O(n)       |
| insert  | O(1)       |
| delete  | O(1)       |
